Mica¶

The mica package provides tools, databases, and file archives to give access to a comprehensive set of information related to Chandra aspect:

  • ACA dark current
  • ACA L0 telemetry
  • ACA diagnostic telemetry
  • ACA HDR3 Pseudo-MSIDS
  • Acquisition star statistics
  • Aspect L1 products
  • Chandra Data Archive and Ocat
  • Guide star statistics
  • Mica.bad_obsids
  • Observation parameters
  • Post-facto aspect V&V
  • Starcheck products
